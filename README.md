# INvoiceBoard
Random Splitter

A simple web application for splitting water bills proportionally among tenants based on headcount.

## Features

- Enter total bill amount and billing period
- Add multiple tenants with names, phone numbers, and number of people
- Automatically calculate proportional shares
- Generate PDF statements
- Share via SMS/WhatsApp or copy message text
- **Local Storage**: Your entered data is automatically saved locally in your browser, so it persists if you accidentally refresh the page or close/reopen the browser.

## Usage

Open `bill.html` in a web browser. No server required for basic functionality (though a local server is recommended for PDF generation due to CORS restrictions).
